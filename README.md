# Paduan menulis markdown

## Heading

``` 
# h1
## h2
### h3
#### h4
##### h5
###### h6
```
## Formating Text
```
**tebal**
*miring*
~~strikeline~~
```
selain menggunakan tanda bintang bisa juga menggunakan garis bawah

```
__text tebal__
_text miring_
```
## Link
```
[deskripsi link](https://google.com)
```
Kita bisa juga menambahkan title untuk tooltips
```
[deskripsi link](https://google.com "google")
```

## membuat list
unordered list
```
* ini
* contoh
* penggunaan 
    * unorder list
```
ordered list
```
1. ini 
2. contoh
3. penggunaan
4. ordered list
```
## menyisipkan kode
